# Generate Templates

To generate a different type of template, `${appName}.${type}.definition.json`, add any of the following options after
`AppThemeTemplateGenerator.ts`:

**Usage**

```commandline
tsx AppThemeTemplateGenerator.ts --vscode
```

|      Options       |
|:------------------:|
|      --icons       |
|      --vscode      |
|      --hyper       |
|      --chrome      |
|       --vim        |
|      --github      |
|     --eclipse      |
|     --jupyter      |
|       --home       |
|     --vsstudio     |
|     --firefox      |
| default: jetbrains |
